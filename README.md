# Installation/Running
## Git Commands:
### Running the web project
1. ```git clone https://github.com/ac111897/AvcolForms.git```
2. ```cd AvcolForms```
3. ```cd AvcolForms.Web```
4. ```dotnet run```
### Running the unit tests
1. ```git clone https://github.com/ac111897/AvcolForms.git```
2. ```cd AvcolForms```
3. ```cd AvcolForms.Tests```
4. ```dotnet test```
